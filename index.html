Task 01

Create & Append Nodes

not started

The DOM is a tree of nodes. The most common are element nodes (like <p>) and text nodes (the words inside them). JavaScript lets you manufacture nodes from scratch and plant them anywhere in the tree.

  1. Inside the task1() function in script.js, use document.createElement() to create a <p> element. Store it in a variable called newPara.
  2. Use document.createTextNode() to create a text node containing any sentence you like. Store it in myText.
  3. Append myText into newPara using .appendChild().
  4. Select the sandbox below using document.getElementById('sandbox-task1') and append newPara into it.
  5. Call markDone(1) at the end of the function so the progress tracker updates.

Task 03

Timed Repeating Commands

not started

setInterval(fn, ms) schedules a function to fire repeatedly every ms milliseconds. It returns an interval ID ‚Äî hold onto it so you can cancel the interval with clearInterval(id) later.

  1. Find the task3() function. Inside, select the element with id="timer-display" and store it.
  2. Declare a variable seconds starting at 0.
  3. Call setInterval() with an arrow function that: increments seconds by 1, then updates the display element's textContent to show the formatted time (see hint). Set the interval to 1000 ms. Store the returned ID as intervalId.
  4. Wire up the Stop button in stopTimer(): call clearInterval(intervalId).
  5. Wire up the Reset button in resetTimer(): stop the interval, set seconds back to 0, and reset the display. Call markDone(3) after starting the timer.

Task 04

Browser Object Model

not started

The Browser Object Model (BOM) is the collection of objects the browser exposes to JavaScript. The top-level object is window ‚Äî the global environment. It contains child objects like window.navigator, window.location, and window.screen that describe the browser and the current page.

  1. Inside task4(), read the following BOM properties and store each in a variable: window.innerWidth, window.innerHeight, window.navigator.language, window.navigator.onLine, and window.location.href.
  2. Select the element #bom-panel. For each value, create a <dl> child using the helper function createBomStat(label, value) already defined in script.js, and append it into #bom-panel.
  3. Bonus: Use window.addEventListener('resize', fn) to keep the viewport dimensions updated live as you resize the browser window.
  4. Call markDone(4) at the end of task4().
